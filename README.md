## Getting Started

<a href="https://nodejs.org/en/"><b>Node</b></a> is required to run this program.

### Prerequisites

npm
```sh
npm install -g npm
```
Dependencies and middleware
```sh
npm install express
```
```sh
npm install body-parser
```
```sh
npm install cors
```

### Installation

Initializing the server
```sh
node server.js
```
Access the localhost on port 8000
```sh
http://localhost:8000/
```
